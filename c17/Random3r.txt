3. Правила применения
=====================

1.  Разместить в рабочем каталоге файлы:
        Random3.c
        Random3.h

2.  Включить в проект файл:
        Random3.c

3.  Добавить в тексты программных модулей директиву:
        #include "Random3.h"

4.  Перед применением генератор надо инициализировать:
        RRand3()
    Начальная установка по RRand3 зависит от системного времени,
    для иной начальной установки можно использовать функцию:
        SRand3 (a,b,c)
    где a, b, c - случайные 32-разрядные числа типа unsigned long.

5.  Для получения случайного 32-разрядного числа unsigned long
    служит функция:
        Rand3();

6.  Для получения случайного числа unsigned long в диапазоне
    0..ULONG_MAX служит функция:
        lRand3();
    Число будет из 32 или 64 разрядов в зависимости от платформы.

7.  Для получения случайного числа double в диапазоне 0..1
    включая 0 и НЕ включая 1 служит функция:
        dRand3();
